You can use next commands to manage the programm.

## Start

```bash
make
```

## Setup

```bash
make build
```

## Run

```bash
make start
```

## Run tests

```bash
make test
```

## Run checkstyle

```bash
make lint
```

## Check update dependencies and plugins

```bash
make update
```
